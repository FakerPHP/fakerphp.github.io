# English (India)

### `Faker\Provider\en_IN\Address`

```php
echo $faker->locality();     // "Vaishali Nagar"
echo $faker->localityName(); // "Kharadi"
echo $faker->societyName();  // "Shanti Apartments"
echo $faker->state();        // "Delhi"
echo $faker->stateAbbr();    // "DL"
```
